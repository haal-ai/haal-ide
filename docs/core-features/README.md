# OLAF Features

This directory contains detailed documentation for each key feature of the Open Lightweight Assistant Framework (OLAF).

## Core Features

- **[OLAF Bootstrap](olaf-bootstrap.md)** - Cross-platform integration for GitHub Copilot, Windsurf, AWS Kiro, and other AI tools
- **[Intent-Based Context Injection](intent-based-context-injection.md)** - Just-in-time context loading based on user intent patterns
- **[Integration with Agents](integration.md)** - Agent-specific setup and configuration for various AI platforms

### Framework Management
- **[Framework Management](framework-management.md)** - XML section tagging and framework validation for better LLM integration
- **[Enhanced Prompt Engineering](prompt-engineering.md)** - Advanced prompt creation, testing, and modernization capabilities
- **[Best Practices](best-practices.md)** - Framework and prompt engineering best practices for optimal OLAF usage

### Interaction Management
- **[Interaction Protocols](interaction-protocols-explained.md)** - Act, Propose-Act, and Propose-Confirm-Act protocols for different safety levels

### Content Generation
- **[Meta-prompting](meta-prompting.md)** - Generate custom prompts following OLAF structure and competency patterns

### Context Management
- **[Memory-map](memory-map-explained.md)** - Project navigation aid and file location mapping for AI agents
- **[Core Principles](core-principles-explained.md)** - Behavioral guidelines and decision-making framework for AI
- **[Chapter Mode](chapter-mode.md)** - Handle long-running processes with context management
- **[OLAF Context Switch](olaf-context-switch.md)** - Switch between different project contexts with template-based system

### Workflow Orchestration
- **[OLAF Workflow Architecture](competency-driven-workflow.md)** - Skills → Competencies → Collections orchestration system
- **[Project Organization](organization.md)** - Skill-based architecture with templates, tools, and structured components
