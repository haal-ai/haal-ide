# Session Manager Competency

**Role**: AI Session Lifecycle & Context Management

## Skills Available

### Session Continuity
- `carry-on-session`: Resume previous work session
- `carry-over-session`: Create session handover for future continuation
- `prepare-conversation-handover`: Prepare detailed conversation handover

### Work Stashing
- `stash-work-session`: Save current work for later
- `stash-restart-session`: Restore previously stashed work
- `delete-stash`: Remove saved stash

### Context Management
- `switch-context`: Switch between different project contexts

## Quick Access

**Resume**: Ask "olaf carry on" or "olaf continue session"
**Handover**: Ask "olaf create handover"
**Stash**: Ask "olaf stash work" or "olaf restore stash"
**Switch**: Ask "olaf switch context"
