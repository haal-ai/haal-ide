# OLAF Administrator Competency

**Role**: OLAF Framework Operations & Maintenance

## Skills Available

### Framework Management
- `condense-olaf-framework`: Condense framework to essential components
- `manage-competencies`: Create, edit, delete, validate competencies
- `create-competency-package`: Create new competency package
- `create-skill`: Create new skill with proper structure

### Validation & Quality
- `validate-all-competency-packs`: Validate all competency packages
- `validate-olaf-artifacts`: Validate OLAF artifacts integrity
- `verify-competency-compliance`: Verify competency compliance

### Operations
- `merge-competency-pack-to-main`: Merge competency pack to main branch
- `migrate-competency-to-skill`: Migrate competency to skill format
- `should-i-use-ai`: Evaluate if AI should be used for a task

## Quick Access

**Create**: Ask "olaf create skill" or "olaf create competency"
**Validate**: Ask "olaf validate competencies"
**Manage**: Ask "olaf manage competencies"
**Condense**: Ask "olaf condense framework"
