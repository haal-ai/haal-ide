# HAAL Deep Dive

HAAL (Human-Agent-Assisted-Learning) provides several key capabilities to enhance AI-human collaboration.

## Core Features

### Platform Integration
- **[HAAL Bootstrap](.olaf/docs/core-features/olaf-bootstrap.md)** - Cross-platform integration for GitHub Copilot, Windsurf, AWS Kiro, and other AI tools
- **[Intent-Based Context Injection](.olaf/docs/core-features/intent-based-context-injection.md)** - Just-in-time context loading based on user intent patterns
- **[Integration with Agents](.olaf/docs/core-features/integration.md)** - Agent-specific setup and configuration for various AI platforms

### Framework Management
- **[Framework Management](.olaf/docs/core-features/framework-management.md)** - XML section tagging and framework validation for better LLM integration
- **[Enhanced Prompt Engineering](.olaf/docs/core-features/prompt-engineering.md)** - Advanced prompt creation, testing, and modernization capabilities
- **[Best Practices](.olaf/docs/core-features/best-practices.md)** - Framework and prompt engineering best practices for optimal HAAL usage

### Interaction Management
- **[Interaction Protocols](.olaf/docs/core-features/interaction-protocols-explained.md)** - Act, Propose-Act, and Propose-Confirm-Act protocols for different safety levels

### Content Generation
- **[Meta-prompting](.olaf/docs/core-features/meta-prompting.md)** - Generate custom prompts following HAAL structure and competency patterns

### Context Management
- **[Memory-map](.olaf/docs/core-features/memory-map-explained.md)** - Project navigation aid and file location mapping for AI agents
- **[Core Principles](.olaf/docs/core-features/core-principles-explained.md)** - Behavioral guidelines and decision-making framework for AI
- **[Chapter Mode](.olaf/docs/core-features/chapter-mode.md)** - Handle long-running processes with context management
- **[HAAL Context Switch](.olaf/docs/core-features/olaf-context-switch.md)** - Switch between different project contexts with template-based system

### Workflow Orchestration
- **[HAAL Workflow Architecture](.olaf/docs/core-features/competency-driven-workflow.md)** - Skills → Competencies → Collections orchestration system
- **[Project Organization](.olaf/docs/core-features/organization.md)** - Skill-based architecture with templates, tools, and structured components
